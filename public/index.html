Setup Your Custom Domain:

  1. Edit config.js and add your domain to the DOMAINS array
  2. Set your domain's MX record to point to this server's IP
  3. Use port 25 for production (set SMTP_PORT=25)
  4. Restart the server

Or set environment variable: MAIL_DOMAINS=yourdomain.com,other.com
